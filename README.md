# 🪂 SkyVault

SkyVault is a modern SaaS platform built for skydivers who want to log, organize, and truly own their jump data.

Most logbooks are either paper-based, outdated, or fragmented across different systems. SkyVault brings everything into one clean, fast, and reliable web app — built with performance and scalability in mind.

This project also serves as a real-world, production-focused SaaS architecture built with Next.js (App Router), TypeScript, and Tailwind.

---

## ✨ What It Does

- 📒 Digital jump logbook
- 🪂 Track jumps, gear, dropzones, aircraft
- 📊 Basic stats and progression metrics
- 👤 User accounts & authentication
- ☁️ Cloud-based, accessible anywhere
- 🧱 Structured for future scaling (teams, DZ management, coaching features)

---

## 🧠 Why This Exists

Skydivers care about precision, progression, and accountability.  
Yet most tools in the space feel stuck in the early 2000s.

SkyVault is an attempt to:

- Build something technically solid
- Design something clean and fast
- Create a foundation for a real SaaS product in the skydiving niche

It’s both a product and a long-term experiment in building software properly.

---

## 🛠 Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** TailwindCSS
- **Architecture:** Server Components by default
- **Planned:** Database integration, auth provider, payments, analytics

## 🚀 Getting Started

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
